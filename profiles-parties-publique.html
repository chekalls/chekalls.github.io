Exemple - Vue publique d'un profil

Cette page illustre comment un profil est montré aux autres utilisateurs lorsque certaines informations sont masquées.

Affichage

  • Les champs marqués publics apparaissent normalement.
  • Les champs privés sont affichés comme "(privé)" ou complètement masqués.
  • Lien "Modifier mon profil" visible uniquement si le visiteur est le propriétaire.
